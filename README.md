# ROBOTS DOOM

ROBOTS DOOM is a top-down survival tank game built with **Unreal Engine 5** and **C++**. The player controls a customizable tank and must survive increasingly difficult enemy waves, collect coins, and purchase upgrades between phases.

The game combines action, strategy, and progression mechanics inspired by arcade survival games. Every phase becomes more challenging, requiring the player to make smart upgrade decisions and use special ammunition and power-ups effectively.

## Features

### Phase-Based Survival

* Infinite waves with increasing difficulty.
* Dynamic enemy spawning.
* Countdown before each phase.
* Victory message after completing a phase.



### Enemy Types

* **Towers** – stationary enemies that rotate and shoot at the player.
* **Robots** – moving enemies that chase and attack the tank.
* **Exploding Robots** – rush toward the player and explode.

### Upgrade Shop

Between phases, players can spend coins to upgrade:

* Health
* Damage
* Movement Speed
* Burst Fire
* Special Ammunition

### Special Projectile Types

* **Normal Shot**
* **Ice Shot** – freezes enemies temporarily.
* **Control Shot** – causes robots to attack other enemies.
* **Explosion Mark Shot** – marks enemies for area damage.

### Power-Ups

Randomly spawned power-ups:

* Shield
* Full Heal
* Damage Boost
* Speed Boost

### UI System

* Start Menu
* Countdown Timer
* Coin Counter
* Ammo Icons and Counts
* Health Bars
* Phase Complete Banner
* Defeat Screen
* High Score Tracking

### Visual Effects

* Niagara particle effects
* Dynamic materials
* Explosion effects
* Shield visual effect

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/NathanLivshitz/BattleBlaster.git
```

2. Open `BattleBlaster.uproject`.
3. If prompted, generate Visual Studio project files.
4. Open `BattleBlaster.sln`.
5. Build the project in Visual Studio.
6. Launch the game from Unreal Engine.
